Projects


View Project

Financial Summary Dashboard

A project where I combined coding + finance. The dashboard lets anyone enter a stock ticker and instantly see 40+ financial metrics, from simple ones like Revenue to more complex ones like Piotroski F-Score.

What makes it special is “Dummy Mode” — a feature that breaks down every metric in simple terms, with formulas and real-world analogies, so even non-finance people can understand.

It’s live on Streamlit Cloud, so anyone can try it out!






View Project

Credit Risk Model

Built a credit risk simulation framework in Python, implementing the Expected Loss model (PD × LGD × EAD) across 10,000+ loans with a 95% confidence interval.

Trained and benchmarked Logistic Regression, Decision Tree, and XGBoost models with AUC/ROC evaluation. Engineered 25+ features from demographic and financial data to improve prediction accuracy.

Project completed as part of an applied credit risk modeling course.






View Project

GPT-2 LLM from scratch

Implemented the GPT-2 architecture from scratch in Python/PyTorch, covering multi-head attention, transformer blocks, and positional encodings.

Leveraged OpenAI’s pretrained weights to validate implementation and perform text generation tasks. Strengthened understanding of transformer internals, transfer learning, and large-scale NLP model design.

This was done by following the book "Build an LLM from Scratch" by Sebastian Raschka
